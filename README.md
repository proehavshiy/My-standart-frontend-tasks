# My-standart-frontend-tasks
This repository is dedicated to my implementations of standard front-end tasks and my attempts to evaluate each of them according to the following criteria:
* Coupling
* Cohesion
* Extensibility
* Maintainability
* Clearability
* Durability
* Performancew
